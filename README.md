# Linkora v2.0

> **A free, offline, open-source link manager** — now with native Windows app & enhanced customization.

![Welcome screen](https://raw.githubusercontent.com/Hyrican/linkora/main/assets/welcome.png)

✨ **What's New in v2.0**
- 🪟 **Native Windows app**: Portable `.exe` built with [Tauri v2](https://tauri.app) — no installation required
- 🎨 **HSL Theme Editor**: Fine-tune your interface with Hue, Saturation, and Lightness sliders (real-time preview)
- 🔁 **Full backward compatibility**: Still works in any browser by opening `index.html`
- 🌐 **12 languages**: Interface available in English, Русский, Español, Deutsch, Français, Italiano, Português, العربية, 中文, 日本語, Polski, Türkçe

---

## ✨ Features

- ✅ **100% offline**: No internet, no tracking, no ads — ever
- 📁 **Organize links**: Create categories, add descriptions, attach screenshots
- 🖱️ **Drag & drop**: Move links between categories intuitively
- 👁️ **Hover previews**: See screenshots and full descriptions on mouse hover
- 🎨 **Custom themes**: Light, Dark, or fully custom HSL color scheme
- 🌍 **Multi-language**: Switch languages instantly, with real-time UI update
- 💾 **Export/Import**: Backup your data as JSON, restore anytime
- ⭐ **Favorites**: Mark important links for quick access
- 🔍 **Smart search**: Filter links by URL or description
